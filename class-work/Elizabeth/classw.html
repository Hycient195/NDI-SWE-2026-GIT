WEEKEND LEARNING EVENT

Learn practical digital skills in one focused weekend.

Join designers, creators and beginner developers for a hands-on event built around real-world projects, collaborative workshops, and guided feedback sessions


At a glance

Date: July 26 to July

Venue: City learning Centre

Format: labs, and guide practice

Audience: Beginners and early-career learners



WHY ATTEND

Build confidence with real tasks instead of only theory.

This event gives attendees time to learn core concepts, ask questions, and practise with guided exercises. By the end of the weekend ,each participant should leave with clearer problem-solving habits and a small body of completed work.


Event goals

4 Learning Tracks
12 Guided Session
80 Student seat

LEARNING TRACKS

Choose a path and build around it.

Design Foundations

Learn layout basis, order choices, spacing systems and simple content hierachy.

Web Basics

Write semantic HTML, style with reusable CSS and create responsive layouts with flexbox.

Career Readiness

Practice presenting projects, improving a profile and explaining what you have built.

Team Collaboration

Work in small groups, response to feedback, and improve ideas through shared decisions.
