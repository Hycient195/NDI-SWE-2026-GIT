Commands

git reset <file>
This is used to remove a specified file from the staging area
git revert <commit>
This is used to create a new commit that undoes the changes made in a specified commit

How to run git revert

add some content git add demo.html git commit -m "Add new line to demo.html" git log to get the commit hashes of the commit git revert "the hash of the previous commit (the commit on the red dot with and X mark)" git add . git commit -m "Revert changes from previous commit" git checkout "the hash of the commit on white dot"

git merge steps

  • Checkout to our own new branch
  • Implement the request feature
  • Commit our changes to our own branch
  • Merge our branch to the main branch
